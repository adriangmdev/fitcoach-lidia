# FitCoach — App de Nutrición con IA

Aplicación web progresiva (PWA) de seguimiento nutricional
con análisis de comidas por cámara usando IA.

## Funcionalidades

- Análisis de macros por foto con Claude AI
- Contador diario de calorías y macros
- Planes de comida editables
- Diseño responsive y PWA instalable

## Tecnologías

Next.js · React · TypeScript · Claude API · Supabase · Tailwind · Vercel
